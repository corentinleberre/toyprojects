# toyprojects

A collection of personal configurations, scripts, and useful tools.

## Contents

| Directory | Description |
| --------- | ----------- |
| `tmux/`   | tmux workspace bootstrapper — per-branch dev sessions with worktree support, Spotify widget, and session timer |
| `ticket/` | One-key Linear ticket creator — opens Claude with a prompt to analyse a pasted screenshot and create a Linear ticket with title, description, and acceptance criteria |
